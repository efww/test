## 자동 테스트 결과

`pnpm test` 기준 전체 테스트가 통과했습니다.

검증된 항목:

- 앱 정보 생성
- Microsoft 로그인 설정 상태
- Microsoft authorize URL 생성
- PDF.js 메모리 바이트 파싱
- PDF 파일 선택 경로 제한
- preload API 범위
- 보호 PDF 후보 감지
- 비PDF 차단
- JSONL 감사 로그 기록
- 저장·인쇄·복사·우클릭·드래그 차단 코드
- 화면 내 저장·다운로드·인쇄 명령 미노출

## 남은 한계

- 실제 회사 Entra ID 앱 승인 전이므로 토큰 발급과 사용자 식별은 검증하지 않았습니다.
- 실제 AIP PDF 복호화와 MIP SDK 연동은 2차 범위입니다.
- 현재 차단은 읽기 전용 UX와 기본 방어선입니다. 화면 캡처 완전 차단 같은 전문 DRM 기능은 범위 밖입니다.
- Windows 테스트 PC 설치 파일 검증은 아직 수행하지 않았습니다. 2차 전 별도 패키징 마일스톤으로 추가할 수 있습니다.

## 2차 진입 조건

아래 조건이 충족되면 2차 PoC로 넘어갈 수 있습니다.

- 회사 Entra ID 앱 등록 또는 관리자 승인
- 테스트 사용자 지정
- 테스트 AIP PDF 제공
- MIP SDK 사용 승인
- 감사 로그 보관 기준 합의
